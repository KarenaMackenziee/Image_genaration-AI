# 🖼️ AI Image Generator using Streamlit

This project is a web-based application built with **Python** and **Streamlit** that generates images from text prompts using AI.

---

## 🚀 Features

- 🎨 Generate images from user input (text prompts)
- ⚡ Fast and simple UI with Streamlit
- 🧠 AI-powered image generation
- 📷 Displays generated images instantly

---

## 🛠️ Technologies Used

- Python
- Streamlit
- PIL (Pillow)
- AI API (Google Generative AI / OpenAI / Stable Diffusion)
